=== Volumes explorer

Volumes Explorer gives you a simple, searchable view of all your volumes in NetApp Workload Factory. Quickly see which volumes serve block or file workloads. View each volume's logical size, physical used capacity, and whether it uses storage efficiency features (compression, deduplication, and compaction). Filter and sort by name, status (available/failed), latency, size, used capacity, and protection features.

=== Workload tagging for volumes and block devices

Workload Factory now identifies and tags volumes and block devices by their primary workload or application. You can assign predefined workload tags such as Microsoft SQL Server or Oracle Database, or create custom tags for your applications. Tags can be applied to individual resources or inherited by connected storage resources. This helps you align storage configurations with workload requirements, understand incident impact, and troubleshoot more effectively.
